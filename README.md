# EduWise – Backend API

Backend for EduWise e-learning platform: authentication, courses, quizzes, certificates, instructor approval, S3 uploads.

## ✅ Why I Built This
This was my **Software Engineering term project** and also my **first large full-stack project after my internship**.  
I wanted to learn real backend engineering, secure auth, file uploads, admin controls, and build a scalable API that talks to a production-ready frontend.

## 🛠 Tech Stack
- **Backend:** Node.js, Express.js
- **Database:** MongoDB + Mongoose
- **Authentication:** JWT + bcrypt
- **Storage:** AWS S3 (video & thumbnail uploads)
- **Deployment:** Heroku
- **Other libraries:**
  - Nodemailer
  - Helmet, CORS (security)

## ✨ Features
- ✅ JWT Login + Register  
- ✅ Roles-based access control: Student / Instructor / Admin  
- ✅ Add & manage courses  
- ✅ Quiz & scoring system  
- ✅ Certificate generation + unique certificate ID  
- ✅ Instructor application + admin approval  
- ✅ Ratings & reviews
